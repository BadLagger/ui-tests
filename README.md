# UI Testing Project

Автоматизированные тесты для веб- и мобильного приложения.

## Структура проекта
```
src/test/java/
├── web/ # Веб-тесты
│ ├── pages/ # Page Object модели
│ ├── tests/ # Тестовые классы
│ └── utils/ # Утилитные классы
├── mobile/ # Мобильные тесты
│ ├── pages/
│ ├── tests/
│ └── utils/
└── resources/ # Ресурсы
```

## Запуск тестов

### Предварительные требования
- Java 17+
- Maven 3.6+
- Chrome браузер

### Запуск всех веб-тестов ###
```
mvn clean test
```

### Запуск через TestNG ###
```
mvn test -DsuiteXmlFile=testng.xml
```

### Запуск из IDE ###

- Откройте testng.xml и запустите как TestNG Suite
- Или запустите класс WebTests как TestNG тест

## Тестовые сценарии ##

### Веб-тесты (8 сценариев) ###
1. Открытие главной страницы
2. Навигация по меню
3. Авторизация с валидными данными
4. Авторизация с невалидными данными
5. Работа с чекбоксами
6. Работа с выпадающим списком
7. Динамическая загрузка элементов
8. JavaScript Alerts
